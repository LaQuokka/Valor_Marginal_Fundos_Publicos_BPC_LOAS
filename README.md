# Valor_Marginal_Fundos_Publicos_BPC_LOAS
Códigos e materiais desenvolvidos durante iniciação científica que visou estimar o Valor Marginal dos Fundos Públicos para o programa Benefício de Prestação Continuada (BPC-LOAS). O projeto está sendo continuado no Laboratório de Economia do Setor Público da FEA-USP. Policy brief será publicado em breve e relatório final publicado (duração da bolsa: set/24 a ago/25).
